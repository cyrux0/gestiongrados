    -- parents the parent classes of this component

    -- declaringClass name of the table definition declaring class (when using inheritance the class that defines the table structure can be any class in the inheritance hierarchy, hence we need reflection to check out which class actually calls setTableDefinition)

    -- tableName database table name, in most cases this is the same as component name but in some cases where one-table-multi-class inheritance is used this will be the name of the inherited table

    -- collate collate attribute
